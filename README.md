# NodeJs-OAuth-Starter


### Installation


```sh
$ git clone https://github.com/shakirck/NodeJs-OAuth-Starter.git
$ cd Habit-Tracker
$ npm install 
$ npm start

Create a key.js file inside the config folder and add fill the following fields

const keys = {

    google:{
        clientID: '<YOUR ID >',
        clientSecret: '<YOUR ID >',
     },
    facebook:{
        clientID:  '<YOUR ID >',
        clientSecret:  '<YOUR ID >',
     },
     gmail:{
         user: '<YOUR ID >',
         pass: '<YOUR ID >'
     }
}

```
